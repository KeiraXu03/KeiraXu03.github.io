**Sept. 2023 – May 2024**

Software development for an automated cart-delivery robot.

- Implemented multi-sensor fusion on **ROS** (motor encoders, IR, etc.) with
  closed-loop pose control via **PID**.
- Built Python and C tooling for real-time sensor analysis and robotic-arm positioning.
- Achieved stable route following with quick disturbance recovery; end-to-end
  travel-distance error within **5%**, with turns handled without overshoot/oscillation.
